## Usage ##

Deserialize the file `Markov.java_serialize` as `Markov` and call `Markov.generate(int maxlength)`

I didn't include the actual dustbin file since it's very large, but you can use `Generate` with any other sqlite file that contains a table `dustbin` with the column `post` containing your set of posts.
